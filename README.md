# SignificantCalculator

一款好用的基礎計算機

## 特別的地方

1. 支援有效小數

2. 使用逆波蘭＋場調度演算法，因此可以直接parse數學式

3. 支援大數，且不會因為數字過長而看不到完整的數值

4. 防呆功能
